# Manga/Manhwa Website Design Guidelines

## Design Approach
**Reference-Based Approach**: Drawing inspiration from modern manga platforms like fl-ares.com, manhwaread.com, and contemporary anime streaming services to create a visually engaging, community-focused experience that appeals to manga and anime enthusiasts.

## Core Design Elements

### A. Color Palette
**Dark Theme Foundation**:
- Primary Background: 15 15% 8% (deep charcoal)
- Secondary Background: 15 12% 12% (slightly lighter charcoal)
- Card Backgrounds: 15 10% 15% (elevated surfaces)

**Brand Colors**:
- Primary Purple: 270 85% 60% (vibrant purple for CTAs and highlights)
- Secondary Pink: 320 70% 65% (complementary pink for accents)
- Success Green: 140 60% 55% (for status indicators)

**Gradients**:
- Hero overlays: Purple to pink diagonal gradients (270 85% 60% to 320 70% 65%)
- Card hover effects: Subtle purple gradient overlays
- Button backgrounds: Linear purple-to-pink gradients

### B. Typography
**Font Stack**: Inter for clean readability
- Headings: 600-700 weight, larger letter spacing
- Body text: 400-500 weight
- Metadata: 300-400 weight, smaller sizes
- Keep hierarchy clear with distinct size differences

### C. Layout System
**Spacing**: Tailwind units of 2, 4, 6, and 8 for consistent rhythm
- Cards: p-4, gap-6 between elements
- Sections: py-8, px-4 for main containers
- Grid gaps: gap-4 for card grids, gap-2 for compact lists

### D. Component Library

**Navigation**:
- Dark header with purple accent gradient on logo
- Search bar with subtle gradient border on focus
- Genre dropdown with modern card-style menu

**Hero Banner**:
- Large featured manga display with gradient overlay
- Rotating carousel with smooth transitions
- CTA buttons with purple gradient backgrounds

**Manga Cards**:
- Clean rectangular cards with rounded corners
- Cover image with gradient overlay on hover
- Rating badges with star icons
- Status indicators (Ongoing/Completed) with color coding
- Chapter count and update timestamps

**Popular Series Sidebar**:
- Compact horizontal cards with thumbnails
- Filter tabs (Weekly/Monthly/All-time) with active purple styling
- Smooth hover animations with purple accent borders

**Latest Updates Section**:
- List-style layout with thumbnail, title, chapter, and timestamp
- Alternating row backgrounds for readability
- Hover states with subtle purple highlighting

### E. Animations
**Minimal and Purposeful**:
- Smooth hover transitions (0.2s ease)
- Card elevation on hover
- Gradient overlay animations on image hover
- NO complex scrolling animations or excessive motion

## Images
**Hero Section**: Large banner image showcasing featured manga artwork with gradient overlay
**Card Thumbnails**: High-quality manga cover art displayed in consistent aspect ratios
**Background Elements**: Subtle abstract patterns or gradients, no distracting imagery

## Key Design Principles
1. **Dark-first design** with strategic use of purple/pink gradients
2. **Content hierarchy** through typography, spacing, and color contrast
3. **Community feel** with ratings, comments, and social proof elements
4. **Modern aesthetics** appealing to anime/manga enthusiasts
5. **Functional beauty** - every visual element serves the user experience
